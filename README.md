# sidi-i-smotri-updates
Сиди и смотри - навайбкоженное проложение, позволяющее бесплатно смотреть фильмы и сериалы. Большинство из них.
Оно дважды пиратское.

Для начала - оно работает на связке Кинопоиск+Фликсбар (или как это там называется)

Для тех, кто не в курсе: берем ссылку на любой сериал/фильм на кинопоиске и заменяем kino на ss:
берем https://www.kinopoisk.ru/film/42571/
меняем на https://www.sspoisk.ru/film/42571/

Это пиратский способ смотреть видео. Зато там можно выбрать кучу доступных переводов.

Второе пиратство заключается в том, что в оригинальном плеере много рекламы, а здесь я постарался ее целиком обрезать.
